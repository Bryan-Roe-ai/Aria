# AI-Assisted Cleanup Agents

This repository includes experimental AI-assisted cleanup tooling.

## Current capabilities

- Detect oversized modules
- Detect TODO-heavy files
- Suggest architectural decomposition
- Recommend archive candidates
- Detect duplicate naming patterns
- Generate governance reports

## Future roadmap

- LLM-driven refactor suggestions
- Automatic module migration plans
- Duplicate code consolidation
- AI-generated architecture documentation
- Autonomous cleanup pull requests
