Production Checklist
  • Model, retopo, and UV in Blender/Maya (check briefs for silhouette).
  • Author PBR textures: basecolor, normal, metallicsmoothness, occlusion, emission.
  • Export FBX to `assets/characters/<id>/model/lod*/` with Unity axes.
  • Run Unity tools to apply import settings, materials, and prefab LODs.
